OSINT Tools
-----------

+ OSINT Framework				--> https://osintframework.com/

+ SpiderFoot					--> C'est un scanner OSINT qui est intégré à kali	 
	
	- spiderfoot -l 127.0.0.1:5001			--> Pour lancer le spiderfoot
	- 127.0.0.1:5001				--> Dans le navigateur pour accéder au GUI
	- New Scan > Scan Name & Target

+ Recon-web					--> C'est un Framework OSINT intégré dans Kali web
	 
+ Recon-ng					--> C'est un Framework OSINT intégré dans Kali CMD

	- recon-ng				--> pour lancer le Framework
	- workspaces create test			--> pour créer un Workplace
	- marketplace search				--> pour voir les modules disponibles sur le marketplace
	- marketplace install [Nom_module]		--> installer un module
	- modules search				--> pour voir les modules disponibles sur la machine
	- modules load [Nom_module]			--> pour lancer un module
	- options set SOURCE [domaine_cible]		--> pour charger une cible on peut utiliser hackxor.net ou h4cker.org comme exemple 
	- run						--> pour lancer une recherche
	- Dashboard					--> pour avoir un résumé


DNS Lookups
-----------

Le Scan DNS permet de trouver les adresses IP utilisé par un domaine ou les sous-domaine

# dnsrecon -d [Domaine cible]				--> commande pour faire le DNS lookups. h4cker.org (domaine exemple)

	Info obtenue : A h4cker.org 185.199.109.153

# dig [Domaine cible] 					--> pour avoir plus d'informations sur le DNS


# dig < domain > mx					--> pour avoir les informations du serveur mail 

# nslookup
	- [Domaine cible]
	- set type=ns
	- [Domaine cible]

┌──(kali㉿Kali)-[~]										--> retourne assez d'information
└─$ nslookup
> server 8.8.8.8
> set type=any
> [Domaine cible]
